## 🚀 Tech Stack

### Frontend
- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🎞️ GSAP (for animations)  
- 🌐 Axios

### Backend
- 🟩 Node.js  
- ⚙️ Express.js  
- 🍃 MongoDB (via Mongoose)  
- 🔐 dotenv  
- 🤖 OpenAI API
